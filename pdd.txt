1. Product Overview

The Wildfire Prediction MVP is a web-based system that allows a user to view recent satellite imagery of a region, run an AI model that predicts wildfire risk or highlights possible fire regions, and visualize the output as a heatmap overlay on a map.
This MVP does not aim for production-level accuracy â€” only a functional end-to-end pipeline.

2. Core Goal for MVP

Build a working prototype that:

Accepts a location/region from user.

Retrieves or accepts satellite image input.

Sends it to an AI inference API.

Returns a predicted fire-risk heatmap or fire/ burned area segmentation mask.

Displays it as an overlay on a web map.

3. Target Users (for MVP)

Students & researchers experimenting with wildfire AI.

Emergency enthusiasts / demo audiences.

Hackathon / portfolio reviewers.
